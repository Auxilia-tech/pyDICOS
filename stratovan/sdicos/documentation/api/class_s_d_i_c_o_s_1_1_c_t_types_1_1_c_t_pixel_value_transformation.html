Detailed Description

Implements CT Pixel Value Transformation, a required single item sequence (0028, 9145) Used for adjusting stored pixel data to the specified output units ( [Pixel Data in Output Units] = [Rescale Slope] * [Pixel Data] + [Rescale Intercept] ) SPECREF V03 Section 7.4.3.3, Table 24.
