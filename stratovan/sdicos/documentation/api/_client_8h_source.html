94  /// Enable SSL connection with the provided public key certificate. Call before connecting to a server.
95  /// Loads a certificate from a.cer, .crt, .p7b, or .pem file.
96  /// This method accepts certificates from files in any of the following formats:

286  enumSopCTTDR = enumSopCT | enumSopTDR, ///< CT and TDR (8-64 bit, 32 bit float)
