Stratovan DICOS Library Utility Classes for the Tag-Level API

Classes used exclusively by the Tag-Level API:

Tags are used to uniquely identify attributes in an attribute manager.

  • SDICOS::Tag
    • Consist of a group number and element number represented as a pair of unsigned shorts. For example (0x0008, 0x0010), where the group number is 0x0008, the element number is 0x0010.

Attributes specify how data is stored. Each attribute class object represents one or more value representations (VR). The toolkit provides the following attribute classes:

  • SDICOS::AttributeString (AE, AS, CS, DA, DS, DT, IS, LO, LT, PN, SH, ST, TM, UI, UT)
    • Represents a variety of string formats (see the string class documentation)
  • SDICOS::AttributeSignedInt16Bit (SS)
    • Represents signed 16-bit data
  • SDICOS::AttributeUnsignedInt16Bit (US)
    • Represents unsigned 16-bit data
  • SDICOS::AttributeSignedInt32Bit (SL)
    • Represents signed 32-bit data
  • SDICOS::AttributeUnsignedInt32Bit (UL)
    • Represents unsigned 32-bit data
  • SDICOS::AttributeSignedInt64Bit (SD)
    • Represents signed 64-bit data
  • SDICOS::AttributeUnsignedInt64Bit (UD)
    • Represents unsigned 64-bit data
  • SDICOS::AttributeFloat (FL)
    • Represents 32-bit floating point data
  • SDICOS::AttributeDouble (FD)
    • Represents double precision data
  • SDICOS::AttributeTag (AT)
    • Represents a tag
  • SDICOS::AttributeOther (OB, OW, OL, OD, OF)
    • Represents a buffer of binary data as 8-bit (OB), 16-bit (OW), 32-bit (OL), 64-bit (OD) or 32-bit floats (OF)
  • SDICOS::AttributeUnknown (UN)
    • Represents data in an unknown format stored as a memory buffer
  • SDICOS::AttributeSequence (SQ)
    • Represents a sequence of items with similar structure. Each item is a set of attributes.
