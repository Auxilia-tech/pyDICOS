bool CreateDX ( )

Indicate that this DX file is intended for viewing. Per the DICOS Specification Section A series of images intended for viewing by an observer, after application of any grayscale transformations specified in the image object such as VOI LUT, shall have an Enumerated Value of "FOR PRESENTATION"

Sets DICOS SOP Class UID (0008, 0016) to "For Presentation" and Presentation Intent Type (0008, 0068) to "FOR PRESENTATION"
