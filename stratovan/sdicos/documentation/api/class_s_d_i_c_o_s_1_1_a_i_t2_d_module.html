Detailed Description

Implements 2D AIT IOD Module SPECREF V03 Section 9.2.2, Table 40

Note: This class is not thread safe DICOS SOP Class Name and UID: Name: "DICOS 2D AIT Storage"
UID: "1.2.840.10008.5.1.4.1.1.501.4"

Derives from ScannerCommon which contains the modules: Owner Object of Inspection Itinerary GeneralScanModule GeneralEquipmentModule SopCommonModule CommonInstanceReferenceModule
