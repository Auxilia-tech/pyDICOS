Stratovan DICOS Library User Level API

The User Level APIs are the highest level APIs provided by the Stratovan DICOS Toolkit. They abstract away the module hierarchy of the DICOS specification that is seen in the Stratovan DICOS Library Tag-level API and Stratovan DICOS Library Module-level API.
The User Level APIs do not require the user to follow the module hierarchy laid out by the DICOS specification when setting attributes, instead they provide methods for a subset of attributes in the DICOS specification and map them to the appropriate module. Examples shown in the Quick Start Section of the Stratovan DICOS Toolkit User Guide utilize the User-Level API. The User Level API makes use of several utility classes for handling 2D and 3D images.

Classes in the User Level API for each modality's Information Object Definition (IOD):

AIT 2D:

  • SDICOS::AIT2D
    This class represents a single AIT 2D scan.

AIT 3D:

  • SDICOS::AIT3D
    This class represents a single AIT 3D scan. It may contain one or more volume sections.

CT:

  • SDICOS::CT
    This class represents a single CT scan. It may contain one or more volume sections

DX:

  • SDICOS::DX
    This class represents a single Digital X-Ray (DX) scan. The DX IOD supports two SOP classes, one for presentation and another for further processing before presentation.

QR:

  • SDICOS::QR
    This class represents a one or more unique chemical compounds.

TDR:

  • SDICOS::TDR
    This class represents a Threat Detection Report (TDR) for an Object of Inspection (OOI). Threat Detection Reports can come from ATR devices, device operators or ground truth assessments.
