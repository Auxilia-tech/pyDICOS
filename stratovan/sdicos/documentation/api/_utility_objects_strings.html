Stratovan DICOS Library String Classes

Strings in the DICOS Specification are represented using various string classes that enforce each value representation's (VR) restriction.

Each string class below has a validation function (IsValid, e.g. SDICOS::DcsTime::IsValid) that returns true or false if the string meets the required restrictions.

  • SDICOS::DcsApplicationEntity (AE)
    • Enforced Restrictions:
      • 16 byte maximum
      • Only allows the default character repertoire ISO-IR 6 excluding "\" and control characters LF (Line Feed 0x0A), FF (Form Feed 0x0C), CR (Carriage Return 0x0D), and ESC (Escape 0x1B)
  • SDICOS::DcsDate (DA)
    • Enforced Restrictions:
      • The string has the format "YYYYMMDD", where YYYY = Year, MM = Month, and DD = Day
  • SDICOS::DcsTime (TM)
    • Enforced Restrictions:
      • The string has the format "HHMMSS.FFFFFF", where HH = Hours, MM = Minutes, SS = Seconds, and FFFFFF = Fraction of a Second
  • SDICOS::DcsDecimal (DS)
    • Enforced Restrictions:
      • 16 bytes max
      • Allows the characters '+', '-', '.', 'e', 'E', and '0' through '9'
  • SDICOS::DcsInteger (IS)
    • Enforced Restrictions:
      • 12 bytes max
      • Allows the characters '+', '-', and '0' through '9'
  • SDICOS::DcsUnlimitedText (UT)
    • Enforced Restrictions:
      • Character maximum of 2^32-2
  • SDICOS::DcsShortText (ST)
    • Enforced Restrictions:
      • Character maximum of 1024
  • SDICOS::DcsLongText (LT)
    • Enforced Restrictions:
      • Character maximum of 10240
  • SDICOS::DcsShortString (SH)
    • Enforced Restrictions:
      • 16 character maximum
      • Excludes all control characters, except ESC (Escape 0x1B)
  • SDICOS::DcsLongString (LO)
    • Enforced Restrictions:
      • Character maximum 64
  • SDICOS::DcsCodeString (CS)
    • Enforced Restrictions:
      • 16 bytes maximum
      • Only allows uppercase letters, '0' through '9', space, and underscore
  • SDICOS::DcsUniqueIdentifier (UI)
    • Enforced Restrictions:
      • 64 bytes maximum
      • Allows the characters '0' through '9' and '.'
  • SDICOS::DcsAge (AS)
    • Enforced Restrictions:
      • The string has the format "NNNT", where N = a number ('0'-'9') and
        T = 'D' for days, 'M' for months, or 'Y' for years.
  • SDICOS::DcsPersonName (PN)
    • Enforced Restrictions:
      • The format for a person's name is "Family Name^Given Name^Middle Name^Prefix^Suffix"
      • The format for an animal's name is "Caretaker Name^Animal Name"
  • SDICOS::DcsDateTime (DT)
    • Enforced Restrictions:
      • The string has the format "YYYYMMDDHHNNSS.FFFFFF&ZZXX", where YYYY = Year,
        MM = Month, DD = Day, HH = Hours, NN = Minutes, SS = Seconds, & = '+' or '-',
        ZZ = Hour offset, and XX = Minute Offset
