Set the LUT Descriptor Attribute : Tag (0028, 3002) using unsigned shorts.

The third value in the LUT Descriptor Attribute, indicating the number of bit for each entry in the LUT Data defaults to 16 bits.

Parameters

nFirstValueMappedFirst Input Value Mapped. Can be signed or unsigned.
