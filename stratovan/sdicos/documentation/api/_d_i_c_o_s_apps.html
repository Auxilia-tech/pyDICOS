(DICOSCert) DICOS Certification Program

Verifies conformance of a DICOS file to the DICOS v03 specification and is certified under certain extra conditions.
DICOS files can be found using the DICOS file's name or a directory name. If no DICOS files are explicitly specified the current directory is searched for DICOS files.

Parameters:
  • -h to display help
    • e.g. -h
  • -cxxx Must be the first parameter. Specify a certification to apply to all DICOS files
    • e.g. -cstratovan
  • -d search for DICOS files in the given folder
    • e.g. -d folder1 folder2 ...
  • -f list the tags for the given file
    • e.g. -f file1.dcs file2.dcs ...
Examples:
DICOSCert -cstratovan -f foo1.dcs foo3.dcs -d bar bar1/foobar -f bar2/foo2.dcs

(DICOSConverter) DICOS File Converter

Convert old DICOS file formats to the current DICOS file format. Provide an input folder with old DICOS files and an output folder for the converted DICOS files. The output folder maintains the folder structure of the input folder.

Parameters:
  • -s Search this folder for older DICOS files
    • e.g. -s "C:\DicosDataOld"
  • -o Output converted DICOS files to this folder
    • e.g. -o "C:\DicosDataNew"
Examples:
DICOSConverter_64R.exe -s "C:\DicosDataOld" -o "C:\DicosDataNew"
