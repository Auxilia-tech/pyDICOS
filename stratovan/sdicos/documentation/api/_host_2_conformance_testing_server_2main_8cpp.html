Server application name that Client connects to. Must by of type SDICOS.DcsApplicationEntity. This parameter corresponds to the Destination Application name property in the SDICOS.Network.DcsClient class and is parameter to the SendOverNetwork() method in the User-Level API objects. e.g. SDICOS.CT.

Accept connections from all source application names to all destination application names. The Source Application Name and Destination Application Name are members of the SDICOS.Network.DcsClient class. They are also passed as parameters to the SendOverNetwork() call in the User-Level API objects. e.g. SDICOS.CT, SDICOS.AIT3D etc.
