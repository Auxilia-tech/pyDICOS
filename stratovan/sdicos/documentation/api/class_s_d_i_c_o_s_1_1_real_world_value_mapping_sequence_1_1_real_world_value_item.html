Real World First and Last Value Mapped (VR based on Mapped Value Representation (0x0060, 0x7006)) Values are used to indicate the first index and last index used with the LUT or first and last value used with slope and intercept (i.e.

[Real Value] = [Slope] * [first value] + [Intercept]) Internally sets Mapped Value Representation (0x0060, 0x7006) to 1 (Signed)
