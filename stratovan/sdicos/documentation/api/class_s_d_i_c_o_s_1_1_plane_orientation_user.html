Set Image Orientation using direction cosines for first row and first column : Tag (0020, 0037), VR = DS [required if Frame Type's, (0008, 9007), first value is "ORIGINAL" and Volumetric Properties, (0008, 9206), is not "DISTORTED"] The coordinate system for CT and DX: The positive Z-axis is the opposite of the scan direction.


The positive Y-axis points towards the ground. Facing the scan direction, the positive X-axis points to the right.
The coordinate system for AIT3D: The positive Z-axis points up. The positive Y-axis points to the front of the scanned person. The positive X-axis points to the left of the scanned person.
