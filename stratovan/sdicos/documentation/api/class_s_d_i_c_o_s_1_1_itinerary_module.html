Detailed Description

Implements Itinerary Module SPECREF V03 Section 4.2, Table 14.

The ItineraryModule is an optional module in the Object of Inspection (OOI) Modules. It is present in CT(EDS), DX(AT), AIT2D, AIT3D, QR and TDR IODs. The Itinerary Module specifies attributes that describe and identify the route and segments of an OOI journey. The Itinerary Module contains attributes that are common for all Objects of Inspection (OOI) owners.
