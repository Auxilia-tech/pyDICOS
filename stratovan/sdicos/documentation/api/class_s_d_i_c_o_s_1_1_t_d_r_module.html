Detailed Description

Implements Threat Detection Report (TDR) Information Object Definition (IOD) SPECREF V03 Section 12, Table 65.

Note: This class is not thread safe DICOS SOP Class Name and UID: Name: "DICOS Thread Detection Report (TDR) Storage" UID: "1.2.840.10008.5.1.4.1.1.501.3"
