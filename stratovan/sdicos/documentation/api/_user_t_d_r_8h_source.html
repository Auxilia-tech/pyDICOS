37 ///
38 /// \class TDR
39 /// \brief TDR represents a Thread Detection Report (TDR) for an Object of Inspection (OOI)
40 ///
41 /// The TDR class is not thread safe.
42 ///
43 /// This class abstracts away the hierarchy of the DICOS specification
44 /// and provides an API that contains methods for setting the attributes needed for a DICOS TDR object.
45 /// The TDR contains information for one or more Potential Threat Objects (PTOs).
46 ///
47 /// Valid default values are provided for many attributes required by the DICOS specification.
48 /// If no PTOs are added to the TDR, make sure to add referenced scans using SDICOS::IODCommon::AddReferencedSopInstance().
49 /// Default values for attributes that can be set through the TDR API are as follows:
50 ///
51 /// - Frame Of Reference UID: Randomized DcsUniqueIdentifier UID string
52 /// - Instance Number: 0
53 /// - Content Date: result of DcsDate::Today() when object is created
54 /// - Content Time: result of DcsTime::Now() when object is created
55 /// - TDR Type: SDICOS::TDRTypes::ThreatDetectionReport::enumUnknownTDRType
56 /// - ATR Manufacturer: Defaults to 'UNKNOWN'. Must be set by user if TDR Type is ATR.
57 /// - ATR Version: Defaults to 'UNKNOWN'
58 /// - ATR Params: Defaults to an empty set.
59 /// - Image Scale Rep: No default value is set. Must be set by user if requirement conditions are met.
60 /// - Alarm Decision Time: Result of DcsDate:Today() and DcsTime::Now() when object is created
61 /// - Alarm Decision: No default value is set. Must be set by user.
62 /// - Abort Flag: Set to SUCCESS. Abort reason is not set.
63 /// - Number of PTOs: Defaults to 0.
64 /// - Number of Alarm objects: Defaults to 0.
65 /// - Additional Screening Performed: Defaults to 'NO'.
66 ///
67 /// - PTO Assessment Information:
68 /// - Assessment Flag : Defaults to UNKNOWN
69 /// - Threat Category : Defaults to Anomaly
70 /// - Assessment Ability : Defaults to No Interference
71 /// - Threat Category Description : Optional attribute. Defaults to empty string.
72 /// - Assessment Probability : Optional attribute. Defaults to -1.0f indicating no value is written to the file.
73 /// - PTO Mass and Density: Defaults to 0 if required and not specified.
74 ///
75 /// - Threat Bounding Polygon: Optional attribute not used by default.
76 /// - Region of Interest Information:
77 /// - PTO Base : Required, no default value.
78 /// - PTO Extents : Required, no default value.
79 /// - PTO Bitmap: Not required.
80 /// - Referenced Instances: Required value must be set by user.
81 /// - If no instance is available, a user message is written to the ErrorLog and writing will fail.
82 /// - Baggage Specific Details:
83 /// - Mass : Defaults to 0.
84 /// - Density : Defaults to 0.
85 /// - Location Description : Defaults to empty string
86 /// - Person Specific Details:
87 /// - Location Description : Defaults to empty string
88 /// - PTO Center : Not used by default.
