Detailed Description

template<class CType>
class SDICOS::Array2DLarge< CType >

Use for large arrays i.e., those requiring more than 100MB of memory.

Memory is allocated in several smaller chunks to improve the chances of allocation at runtime. Array items are accessed in reverse coordinate order in order to improve cache access patterns. Thus, you must access the Y direction first, followed by the X direction to access the array element.
