Dynamic Screening Flag : Tag (0060, 7010) VR = CS [optional] Defines the inspection mode.

Dynamic screening is an adaptive Risk Based Screening (RBS) security screening capability that provides personalized screening by adjusting screening parameters based on what is known about each passenger on the day of travel.It enables dynamic switching of automated detection algorithms for the screening of passengersand their belongings(including accessible property and checked baggage) based on passenger risk designations in a real-time fashion.

Parameters
nFlagenum value to set the dynamic screening flag to, Unknown enumerations map to enumNone.
Returns
true if the flag is successfully set, false otherwise
