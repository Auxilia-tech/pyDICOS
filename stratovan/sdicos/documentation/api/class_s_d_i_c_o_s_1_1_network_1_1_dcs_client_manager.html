Detailed Description

Handles sending DICOS files to all managed clients. This class maintains a buffer to reduce allocation/deallocation times incurred by sending the same file to multiple targets. The user can provide and reclaim this buffer.

Perform a C-Store operation for each managed client.

Call ConnectToServers() before calling this function. Handles starting and stopping DICOS sessions if all clients have no sessions started. If not all clients have active sessions, a session will start for those without active sessions and this function will not stop any of the sessions. If all clients have active sessions, those session reamin active when this function exits.
