Iterator& SDICOS::Array3DLarge< CType >::Iterator::operator++ ( )

The prefix operator++ increments the Iterator one step in the iteration direction.

ITERATION_DIRECTION_FORWARD: move to a slice with greater depth ITERATION_DIRECTION_BACKWARD: move to a slice with lesser depth

If the Iterator is at the iteration itEnd value operator++ does nothing.

Dereferencing the Iterator return a pointer to the current slice or NULL.

Returns
a reference to the Iterator is returned
