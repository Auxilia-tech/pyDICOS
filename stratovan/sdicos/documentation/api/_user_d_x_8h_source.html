46 ///
47 /// \class DX
48 /// \brief DX represents a single DX/X-Ray Scan. This class is not thread safe.
49 ///
50 /// This class is not thread safe.
51 /// It abstracts away the hierarchy of the DICOS specification
52 /// and provides an API that contains methods for setting the attributes needed for a DICOS DX object.
53 /// This class implements a DX IOD that contains a single Digital X-Ray image.
54 /// The image consists of unsigned 16/32/64-bit integer or float data and an optional 16-bit LUT if the IOD is for presentation.
55 ///
56 /// The default presentation intent type is 'FOR PROCESSING'.
57 /// The default SOP class UID is set to SOPClassUID::GetDXProcessing().
58 /// Default image position is (0,0,0) and image orientation is X = (1,0,0) and Y = (0,1,0).
59 ///
60 /// Default values for each attribute are documented with the set and get functions
61 ///
62 /// DX/X-Ray Specific Modules
63 /// - Modules in the DICOS DX/X-Ray Modality specification, are implemented as base classes of SDICOS::DX
64 /// - XRay Equipment, DX Detector, DX Positioning, XRayGeneration, and XRay Filtration
65 ///
