Stratovan DICOS Library Utility Classes for the User-Level API

Classes used exclusively by the User-Level API:

  • SDICOS::Image2D
    The Image2D class is used to store 2D signed and unsigned data for IODs that need it.
  • SDICOS::Volume
    The Volume class is used to store 3D signed and unsigned data for IODs that need it.
  • SDICOS::IODCommon
    This class is a base class the holds attribute common to all IODs. The High Level API classes derive from this class and specialize it with IOD-specific attributes.
  • SDICOS::SectionCommon
    This class is a base class that holds attributes common to all types of scanned data. IOD specific Section classes such as SDICOS::Section
    derive from this class.
