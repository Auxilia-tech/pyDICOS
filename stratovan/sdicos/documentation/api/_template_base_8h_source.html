45 ///
46 /// \class IODCommon
47 /// \brief Base class for Information Object Definition (IOD) Template classes containing shared, required attributes.
48 ///
49 /// Notes:
50 /// 1) The IODCommon class contains attributes that are common to all IODs.
51 /// 2) In general, only required attributes are supported as part of the API.
52 /// 3) Template classes for specific IODs derive from IODCommon and implement IOD specific attributes.
53 /// 4) Default values for most string attributes are set to 'UNKNOWN'.
54 /// 5) Default values for date and time attributes are set to DcsDate::Today() and DcsTime::Now().
55 /// 6) Bad or non-conformant values encountered during the reading/loading of a DICOS file are ignored and replaced with the default value.
56 ///
57 /// Notes:
58 /// Only required attributes in the SOP Common Module are supported in the API.
59 /// No API is provided for the Acquisition Context Module because all attributes are under a type 2C (Mandatory but NULL permissible) Sequence. Module handled by deriving class.
60 /// Limited API is provided for the Common Instance Reference. Other module functionality handled by deriving class.
61 ///
62 /// Provides API for:
63 /// - OOI Owner
64 /// - Itinerary Module
65 /// - Object of Inspection (OOI)
66 /// - General Scan Module
67 /// - General Series Module
68 /// - General Equipment
69 /// - SOP Common Module
70 /// - Common Instance Reference Module (Getter functions only)
71 ///
72 /// No API provided for:
73 /// - Acquisition Context Module
74 /// - Common Instance Reference (other than Getters)
75 ///
