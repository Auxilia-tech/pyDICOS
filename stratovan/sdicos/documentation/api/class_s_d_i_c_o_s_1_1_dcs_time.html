SDICOS::DcsTime Class Reference

This string represents a time. The string follows the format "HHMMSS.FFFFFF", where HH = Hours, MM = Minutes, SS = Seconds, and FFFFFF = Fractional Seconds The string also supports time ranges in the following formats: "HHMMSS.FFFFFF-" = Specifies start time "-HHMMSS.FFFFFF" = Specifies end time "HHMMSS.FFFFFF-HHMMSS.FFFFFF" = Specifies start and end time. Detailed Description
