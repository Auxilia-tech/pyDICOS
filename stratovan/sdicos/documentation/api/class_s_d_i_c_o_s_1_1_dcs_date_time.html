SDICOS::DcsDateTime Class Reference

This string combines date and time into a single string. The string has the format "YYYYMMDDHHNNSS.FFFFFF&ZZXX", where YYYY = Year, MM = Month, DD = Day, HH = Hours, NN = Minutes, SS = Seconds, & = '+' or '-', ZZ = Hour offset, and XX = Minute Offset The string has the following range formats: "-YYYYMMDDHHNNSS.FFFFFF&ZZXX" "YYYYMMDDHHNNSS.FFFFFF&ZZXX-" "YYYYMMDDHHNNSS.FFFFFF&ZZXX-YYYYMMDDHHNNSS.FFFFFF&ZZXX".
