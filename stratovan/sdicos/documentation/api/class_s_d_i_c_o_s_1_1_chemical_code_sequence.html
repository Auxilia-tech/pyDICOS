Detailed Description

Implements a specialized code sequence for chemical compounds with context ID set to "DCS1" and scheme designator set to "IUPC" SPECREF V03 Section 17.3.1, Table 133.

Set the chemical compound following the InChI format.

Parameters
strInChIKeyMust be set to a 27 character key generated by InChI software. (Extended Code Value (0008, 0101) [required])
strInChICompoundMust be set to the full standard InChI value of the compound. String starts with "InChI=1S/", where "1" is the InChI version number and "S" indicates standard InChI. (Extended Code Meaning (0008, 0104), VR = LT [required])
