Detailed Description

Implements DX IOD SPECREF V03 Section 8.2.1, Table 33.

Note: This class is not thread safe DICOS SOP Class Name and UID (For Processing):
Name: "DICOS Digital X-Ray (DX) Image Storage-For Processing" UID: "1.2.840.10008.5.1.4.1.1.501.2.2"

DICOS SOP Class Name and UID (For Presentation):
Name: "DICOS Digital X-Ray (DX) Image Storage-For Presentation"
UID: "1.2.840.10008.5.1.4.1.1.501.2.1"
